Smart Field Display by Status

The edit sheet adapts based on roll status - showing only relevant fields:

  • Loaded: Camera, Film, Status, Loaded Date, Notes + Quick Actions (Mark as Lab, Archive)
  • Fridge: Same as Loaded (no dates tracked for cold storage)
  • Lab: All base fields + Lab Date
  • Scanned: All previous + Scanned Date
  • Processed: All previous + Processed Date
  • Archived: All fields + Archived Date

Empty date fields show "Not set" in gray, available to tap and set.
