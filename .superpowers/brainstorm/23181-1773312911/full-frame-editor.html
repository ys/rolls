Full Frame Notes Editor Features

  • Access from edit sheet — "EXPAND" button next to Notes label
  • Full screen editing — Back/Done header, large textarea takes full height
  • Markdown support — Write structured notes with headings, lists, bold/italic
  • Character count — Bottom-right corner shows length
  • Optional toolbar — Quick markdown formatting buttons (heading, list, bold, etc.)
  • Optional preview tab — Switch between Edit/Preview to see rendered markdown
  • Auto-save on Done — No explicit save button, done = save and return

When to Use Full Frame Editor

Perfect for:

  • Loaded rolls where you're planning shots
  • Lab rolls where you're documenting what you shot
  • Any roll where you need more than 1-2 lines of notes
