Key Design Elements

  • Pure black background (#000) — true darkroom feel
  • No rounded corners — sheets, cards, buttons all sharp-edged
  • Minimal borders — only where needed for structure (1px #222)
  • Left border accents — status, emphasis via 2px colored left borders
  • Amber (#fbbf24) safelight — only active nav items + FAB
  • Typography hierarchy — weight, size, spacing (no color decoration)
  • Flat bottom nav — no glassmorphism, just 1px top border
  • Underline inputs — single bottom border, no backgrounds
  • Uppercase labels — tiny, spaced caps for field labels
